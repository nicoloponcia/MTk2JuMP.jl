# Examples

This folder contains runnable usage examples for MTk2JuMP.

Guidelines:
- Keep example scripts in this folder (not in src).
- Use clear names such as quickstart_<feature>.jl and advanced_<feature>.jl.
- Keep examples aligned with current exported APIs.

Available examples:
- quickstart_LGR.jl: computes LGR collocation matrices and plots Lagrange basis functions.

Run with:
- julia --project=. examples/quickstart_LGR.jl
